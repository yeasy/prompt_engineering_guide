# 第十章：多模态提示工程

## 10.2 图像理解与视觉提示

视觉语言模型（VLM）可以理解图像内容并回答相关问题。本节介绍图像提示的设计方法。

### 基本图像提示模式

#### 1. 描述任务

```
请详细描述这张图片的内容，包括：
- 主要对象和场景
- 颜色和构图
- 氛围和风格
```

#### 2. 问答任务

```
关于这张图片：
1. 图中有多少人？
2. 他们在做什么？
3. 这是什么场合？
```

#### 3. 分析任务

```
请分析这张商品图片：
- 产品外观和设计特点
- 展示的使用场景
- 目标消费群体判断
```

### 图像提示技巧

#### 明确指向

```
请关注图片左上角的文字内容...
请分析图片中央人物的表情...
```

#### 结构化输出

```
分析这张图表，以JSON格式输出：
{
  "chart_type": "图表类型",
  "title": "标题",
  "key_data": [],
  "trend": "趋势描述"
}
```

#### 多图比较

```
比较这两张产品图片：
图片1：[第一张图]
图片2：[第二张图]

请分析：
1. 外观设计差异
2. 功能特点对比
```

### 常见应用场景

- OCR与文档理解
- 图表数据提取
- 产品图片分析
- UI界面评估
- 图像质量检测

### 小结

图像提示是多模态 AI 应用的核心技能。有效的图像提示需要明确任务目标（描述、问答、分析）、精确指向图像内容区域，并根据下游需求设计适当的输出格式。在实际应用中，图像理解已广泛用于 OCR 文档处理、图表数据提取、产品图片分析、UI 界面评估等场景。

随着 VLM 模型能力的不断提升，图像提示工程将成为越来越重要的技能。建议读者多实践不同场景，积累针对性的提示模板库。
