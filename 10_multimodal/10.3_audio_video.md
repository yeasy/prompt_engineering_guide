# 第十章：多模态提示工程

## 10.3 音频与视频处理

支持音频和视频的多模态模型能够处理更丰富的信息形式。本节介绍相关的提示设计方法。

### 音频处理

#### 语音转录

```
请将这段音频转录为文字：
- 保留语气词和停顿标记
- 标注说话人切换
- 识别音乐或环境音
```

#### 音频分析

```
分析这段音频：
1. 内容摘要
2. 说话人情绪判断
3. 背景环境推断
```

### 视频处理

#### 视频摘要

```
请观看这段视频并生成：
1. 内容摘要（200字以内）
2. 关键时间点列表
3. 主要人物和场景
```

#### 视频问答

```
根据视频内容回答问题：
问题：视频中演讲者的主要观点是什么？

请基于视频内容回答，标注相关时间点。
```

### 视频提示技巧

- 分段处理长视频
- 关注关键帧而非每一帧
- 结合音频和画面信息
- 指定需要关注的时间范围

### 小结

音频和视频提示需要考虑时间维度，合理设计任务粒度，并综合利用多种信息来源。
