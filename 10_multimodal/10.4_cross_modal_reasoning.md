# 第十章：多模态提示工程

## 10.4 跨模态推理与融合

跨模态推理是指综合利用多种模态的信息进行分析和推理。本节介绍相关技巧。

### 跨模态融合策略

#### 信息互补

```
分析这张产品图片和下方的产品描述文字：

[图片]

产品描述：{text_description}

请综合图文信息分析：
1. 图片与描述是否一致？
2. 有哪些信息只在图片/文字中出现？
3. 综合两者的完整产品理解
```

#### 多源验证

```
用户提供了两个信息来源：
1. 产品照片
2. 用户评价文本

请分析评价内容是否与图片展示相符，指出可能的不一致之处。
```

### 复杂多模态任务

#### 图文匹配

```
判断这张图片与以下哪个描述最匹配：
A. {描述1}
B. {描述2}
C. {描述3}

请说明选择理由。
```

#### 视觉推理

```
观察这张图片，回答问题：
如果图中的物体继续向右移动，接下来可能发生什么？

请基于图片内容进行推理。
```

### 小结

跨模态推理充分利用不同模态的优势，提示设计需要明确模态间的关系和综合分析的目标。
