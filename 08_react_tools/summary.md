# 第八章：ReAct 与工具使用

## 本章小结

本章介绍了ReAct 框架和工具使用技术，这些技术使语言模型能够与外部世界交互，大大扩展了其应用能力。

### 关键概念

- **ReAct**：推理与行动相结合的提示框架
- **函数调用**：模型生成结构化的工具调用请求
- **外部知识源**：搜索引擎、知识库、数据库等
- **智能体（agent）**：能自主规划和执行任务的AI系统

### 核心要点

1. **ReAct循环**：Thought → Action → Observation → 循环
2. **函数调用设计**：功能单一、描述清晰、结构化返回
3. **知识源接入**：权衡时效性、权威性和安全性
4. **Agent设计**：规划、执行、反思、记忆四大能力

### 实践要点

- 工具功能应单一明确
- 参数和返回值需结构化
- 多工具协作需要编排策略
- Agent需要明确的角色和流程定义

5. **实战练习核心**
   - 模拟 Thought-Action-Observation 循环
   - 编写结构化的工具 Schema
   - 设计工具调用的容错提示词
